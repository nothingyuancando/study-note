## 数字电路仿真

Virtual Circuit Board

这是一个 Steam 上的游戏，可以自由的搭建数字电路。2022年5月发售，原价50rmb。

这个游戏逻辑门需要自己画，任何形状都可以，因为形状可以随便搞，所以需要指定输入脚和输出脚：

输入（read）、输出（write）、信号线（traces）、跳线（cross）

在基础阶段，了解这些就可以玩这个游戏了，在学完计算机原理等知识后，可以用这个游戏尝试去搭一个简单的CPU。
